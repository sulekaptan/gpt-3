# GPT-3

## Run Locally 

Clone the project  

~~~bash  
git clone https://github.com/sulekaptan/gpt-3.git
~~~

Go to the project directory  

~~~bash  
cd gpt-3
~~~

Install dependencies  

~~~bash  
npm install
~~~

Start the project  

~~~bash  
npm start
~~~
